# Portfolio de Ciencia de Datos y Análisis Cuantitativo

¡Hola! Bienvenido/a a mi repositorio principal.

Soy estudiante de **Ciencia de Datos en la Universidad Complutense de Madrid (UCM)**. Este espacio es un portafolio vivo donde recopilo, documento y comparto los proyectos analíticos y de modelado que voy desarrollando a lo largo de mi formación académica y trayectoria profesional.

Mi enfoque principal es transformar datos complejos en conocimiento accionable, aplicando rigor estadístico, algoritmos predictivos y técnicas de análisis multivariante para resolver problemas y extraer patrones de valor.

## Stack Tecnológico y Habilidades
A lo largo de mis proyectos, aplico y combino diversas herramientas y metodologías analíticas:

* **Lenguajes de Programación:** Python, R, SAS, SQL.
* **Análisis y Modelado:** estadística descriptiva e inferencial, análisis multivariante, series temporales, econometría, análisis espacial.
* **Machine Learning:** modelos de clasificación y regresión, aprendizaje supervisado y no supervisado (Clustering, Random Forest, XGBoost, ...).
* **Ingeniería de Datos (ETL):** limpieza de bases de datos relacionales, manejo de valores atípicos (*outliers*), lectura y procesamiento de ficheros planos masivos.
